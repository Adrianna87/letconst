What is the difference between var and let?
var can be redeclared and reassigned. let can be reassigned but not redeclared.

What is the difference between var and const?
var can be redeclared and reassigned. const cannot be reassigned nor can it be redeclared

What is the difference between let and const?
let can be reassigned but not redeclared. const can neither be reassigned nor redeclared.

What is hoisting?
Variable declared with 'var' is are run first, before any initialization. The variables 'let' and 'const' are not hoisted. They cannot be accessed beofre initialization. You can declare with 'let' without giving it a value but you cannot declare 'const' without giving it a value.
